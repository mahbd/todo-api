# ToDo

This is backend for ToDo app. It is written in Python using Django framework. This provides REST API for Web, Desktop
and Mobile clients.

## Available API

1. User
    1. Login
    2. Logout
    3. Register
    4. Change Password
    5. Reset Password
    6. Token Login
    7. Token Refresh
    8. User Details
    9. User Update
2. ToDo
    1. Create ToDo
    2. Update ToDo
    3. Delete ToDo
    4. Get ToDo
    5. Get ToDo List
    6. Set Alarm
    7. Set reminder
    8. Share ToDo
    9. Get suggestions
